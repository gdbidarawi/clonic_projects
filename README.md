# 🏥 Clinic Management System

A beginner-friendly **Clinic Management System** built with Python using file handling. No external libraries required.

---

## 📋 Features

- ✅ Add Patients
- ✅ View Patients
- ✅ Add Doctors
- ✅ View Doctors
- ✅ Book Appointments
- ✅ View Appointments
- ✅ Data saved to text files
- ✅ Simple menu-driven interface

---

## 📁 Project Structure

```
clinic-management-system/
│
├── main.py             # Main program file
├── patients.txt        # Patient data storage (auto-created)
├── appointments.txt    # Appointment data storage (auto-created)
├── doctors.txt         # Doctor data storage (auto-created)
├── README.md           # Project documentation
└── requirements.txt    # Dependencies (none required)
```

---

## ⚙️ Technologies Used

- **Python 3** — Core programming language
- **File Handling** — Data stored in `.txt` files
- **No external packages** — Runs with standard Python only

---

## 🚀 How to Run

### Step 1: Install Python

Download from: https://www.python.org

### Step 2: Clone or Download the Project

```bash
git clone https://github.com/YOUR_USERNAME/clinic-management-system.git
cd clinic-management-system
```

### Step 3: Run the Program

```bash
python main.py
```
